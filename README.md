# A simple package for interacting with Hino's API


### Installation
```sh
npm i hino.api
```
```sh
yarn add hino.api
```

### Usage
```js
const API = require("hino.api");

API.connect();
```

### Options

| Options | Description |
| - | - |
|version| get API's name|
url| get API's url|
| latency | get the API's latency ping|
| status | get the API's connection status|
